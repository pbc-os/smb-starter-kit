# SMB Email Templates

Ready-to-adapt templates for the most common small business emails. The agent should customize these based on the specific business, brand voice, and context.

## Customer Service

### Order Confirmation
```
Subject: Your order is confirmed — [Business Name]

Hi [Name],

Thanks for your order! Here's what we've got:

[Order details]

Expected [pickup/delivery]: [date/time]

Questions? Just reply to this email.

Thanks,
[Business Name]
```

### Complaint Response
```
Subject: RE: [Original subject]

Hi [Name],

I'm sorry about this. That's not the experience we want you to have.

Here's what I'm doing to make it right: [specific action]

[If applicable: refund/replacement/discount details]

I'll follow up by [date] to make sure everything's resolved.

[Name]
[Business Name]
```

### Review Request (post-purchase)
```
Subject: How was your experience?

Hi [Name],

Thanks for [shopping with us / your recent order]. We'd love to hear how it went.

If you have a minute, a quick review would mean a lot to our small team: [review link]

Thanks for supporting local!

[Business Name]
```

## Vendor Communication

### New Vendor Inquiry
```
Subject: Inquiry from [Business Name] — [product/service]

Hi,

I'm [Name] from [Business Name] in [City]. We're a [brief description] looking for [what you need].

Could you send us:
- Product catalog / price list
- Minimum order quantities
- Delivery schedule for [area]
- Payment terms

Our typical order volume would be [range].

Thanks,
[Name]
[Business Name]
```

### Payment Notification
```
Subject: Payment sent — Invoice #[number]

Hi [Name],

Payment for invoice #[number] ($[amount]) was processed today via [method].

Please confirm receipt when you get a chance.

Thanks,
[Name]
```

### Order Placement
```
Subject: Order — [Business Name] — [date]

Hi [Name],

Please process the following order for delivery on [date]:

[Item list with quantities]

Delivery to: [address]
Contact on delivery: [name, phone]

Please confirm receipt and any substitutions.

Thanks,
[Name]
```

## Internal / Team

### Schedule Update
```
Subject: Schedule update — week of [date]

Team,

Here's the updated schedule for next week:

[Schedule details]

Changes from last week:
- [Change 1]
- [Change 2]

Let me know ASAP if you have any conflicts.

[Name]
```

### Policy Update
```
Subject: Update: [Policy topic]

Team,

Starting [date], we're updating our [policy topic]:

What's changing:
- [Change 1]
- [Change 2]

Why:
[Brief explanation]

The updated SOP is here: [Drive link]

Questions? Let's discuss at standup.

[Name]
```
